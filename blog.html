DOM stands for Document Object Model. DOM is a kind of Programming API for HTML document. In browser every html page consider as a document. Browser give output of a html document like js object. It's called array like object. By manipulating DOM (Get, Create, Update, Delete element) it possible to make a webpage dynamic with help of javaScript. With out DOM Manipulation the website will stay static.
